Project Title
Telecom Customer Churn Analysis Dashboard

Project Overview:
This project analyzes telecom customer churn behavior using Python for data cleaning and Power BI for visualization. The objective was to identify major factors contributing to customer churn and provide actionable business insights.

Tools Used:
Tools & Technologies
Python
Pandas
Power BI
Excel

Data Cleaning Steps:
removed duplicates
handled blank values
converted datatypes
transformed SeniorCitizen values
renamed columns
exported cleaned dataset

Dashboard KPIs:
Total Customers
Churned Customers
Churn Rate
Average Monthly Charges
Key Insights

Key Business Insights:
Customers with month-to-month contracts showed the highest churn.
Customers with high monthly charges were more likely to churn.
Most churn occurred within the first 12 months.
Electronic check users had higher churn behavior.
Fiber optic internet customers showed elevated churn rates.
